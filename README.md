# Vehicle Rental Management System

## Overview

Vehicle Rental Management System is a web-based application developed using ASP.NET Web Forms. The system allows users to browse available vehicles, view vehicle details, and make rental bookings for a selected duration.

The application also provides administrative functionalities for managing vehicles, bookings, registrations, brands, fuel types, and accessories.

---

## Features

### User Features
- User Registration
- Vehicle Browsing
- Vehicle Booking
- Rental Duration Selection
- Vehicle Details View

### Admin Features
- Manage Vehicles
- Manage Vehicle Brands
- Manage Fuel Types
- Manage Accessories
- Manage Customer Registrations
- Manage Bookings

## How to Run

1. Open the project in Visual Studio 2013.
2. Restore the database file from the App_Data folder.
3. Configure the connection string in Web.config.
4. Build the project.
5. Run using IIS Express or Local IIS.
